Các bước chạy server :
    Bước 1 :
        python3 -m vevn venv (linux)
        python -m venv venv (window)
        (kiểm tra venv đã hoạt động chưa trước khi sang bước tiếp theo)
    Bước 2 :
        pip install -r requirments.txt
    Bước 4 :
        python3 main.py

database trong db_service.py 

Chạy lệnh "python3 test.py" trên một terminal khác để  lấy dữ liệu về  database (kiểm tra IP của máy trước khi chạy lệnh)
